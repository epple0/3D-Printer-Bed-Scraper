# 3D-Printer-Bed-Scraper
A 3D printable bed scraper that helps remove prints from the build plate with control and minimal force. Helps uses with delicate prints and a 3D printer.

Summary: This tool helps remove prints from a build plate of a 3D printer. The design is focused on the blade and the grip, with easy printability.
